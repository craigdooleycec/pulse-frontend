What is CEC Pulse?

Developed by the HSCNI Clinical Education Centre, Pulse is a powerful tool to help educators socialise clinical learning. Check the pulse of your learners' opinions, priorities and challenges.

How does it work?

Create a Pulse by adding a question and three response options. Embed the Pulse into your Articulate Rise course for a seamless social learning activity.

FAQ: how do I copy the embed code?

When you create a Pulse, the embed code is automatically copied. Simply paste the code into an "embed" block in Articulate Rise.

FAQ: I lost my embed code

Go to the Manage Pulses area, find your Pulse, and select the "Copy embed" option.

FAQ: how do I edit my Pulse

Go the Manage Pulses area, find your Pulse, and select the "Edit" option.

FAQ: can I check what the current pulse (results) are?

Yes! In the Manage Pulses area, select the "Pulse rate" option.
